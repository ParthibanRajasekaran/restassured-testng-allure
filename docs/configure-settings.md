### Configure settings.xml for Maven downloads:

1. Download the settings.xml file from Project directory
2. Copy this file to your maven root folder as specified below

#### MAC :

- Right click finder -> go to folder -> ~/.m2/

#### Windows :

- C:\Users{username}.m2
